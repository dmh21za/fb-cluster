# fb-cluster
Revising how to perform clustering using downloaded Facebook posts.

# Instructions

# Getting the data
1. Go to 'https://business.facebook.com/latest/insights/content'
2. Click Export Data (towards the top)
3. Select your page, date range, ALL the metric presets, and Content Level = Post

# Setup
1. Create a .env file with variables as shown in .env.example
2. Move your Facebook data into a folder which corresponds with the DATA_DIR in .env.
